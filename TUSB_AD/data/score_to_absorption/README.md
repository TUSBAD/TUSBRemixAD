### ※このデータパックは [AIAkaishi](https://github.com/Ai-Akaishi) 様のデータパック [ScoreToHealth](https://github.com/Ai-Akaishi/ScoreToHealth) をAbsorptionAmount用に改変したものです。

# ScoreToAbsorption

プレイヤーの緩衝体力をスコアで設定できる

# attribute UUID
``00000002-0000-0000-0001-00`` ～ ``00000002-0000-0000-0001-16``
この範囲を使用しています
※16進数表記

# 動作確認済みバージョン / Verified minecraft versions

- 1.20.4

# 使い方/How To Use

プレイヤーのScoreToAbsorptionに、目的のAbsorptionAmountの100倍をセットする 
Simply set the player's ScoreToAbsorption to 100 times the desired AbsorptionAmount.

例)12.34にしたい場合  
/scoreboard players set @s ScoreToAbsorption 1234
